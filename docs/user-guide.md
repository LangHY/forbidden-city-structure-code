> 故宫主题沉浸式交互网站 - 用户操作指南

## 一、快速开始

### 1.1 在线访问

打开浏览器，访问项目部署地址即可开始体验。或者，直接访问 https://replicate-website-effect.vercel.app

### 1.2 本地运行

#### 环境要求

| 要求 | 版本 | 检查命令 |
|------|------|---------|
| Node.js | 18.0+ | `node -v` |
| npm | 9.0+ | `npm -v` |
| 浏览器 | Chrome 90+ / Firefox 88+ / Safari 14+ | - |

#### 安装步骤

```bash
# 1. 进入项目目录
cd replicate-website-effect

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev

# 4. 打开浏览器访问
# http://localhost:5173
```

## 二、功能介绍

### 2.1 开场页 (Opening)

**入口路径：** `/`

**功能描述：**

电影风格的沉浸式开场页面，展示故宫文化意境。

#### 操作方式

| 操作          | 效果              |
| ----------- | --------------- |
| **向下滚动**    | 触发视频定格动画，过渡到路由页 |
| **点击右上角图标** | 切换亮/暗主题         |
| **等待 10 秒** | 自动生成新的 AI 诗句    |



#### 动画阶段

| 阶段 | 滚动进度 | 效果描述 |
|------|---------|---------|
| 1 | 0% - 50% | 正常显示开场页 UI |
| 2 | 50% - 70% | 诗句、准星开始淡出 |
| 3 | 70% - 85% | 背景网格模糊，视频层模糊 |
| 4 | 85% - 100% | 路由页由模糊渐变为清晰 |

## 三、主题切换

### 3.1 亮色主题 (Light)

```
背景色：#faf9f7 (米白色)
主色调：#b91c1c (故宫红)

装饰色：#78716c (灰色)
```

**适用场景：**
- 日间浏览
- 打印输出
- 高对比度需求

```
背景色：#1a1a1a (深黑色)

文字色：#faf9f7 (白色)
装饰色：#a8a29e (浅灰色)
```

**适用场景：**

## 六、常见问题

### Q1: 页面加载很慢怎么办？

**可能原因：**
- 网络连接不稳定
- 3D 模型文件较大（每个约 5-10MB）

**解决方案：**
1. 检查网络连接
2. 等待资源预加载完成
3. 使用更快的网络环境

### Q2: 3D 模型无法显示？

**可能原因：**
- 浏览器不支持 WebGL 2.0
- 显卡驱动过旧

**解决方案：**
1. 更新浏览器到最新版本
2. 更新显卡驱动
3. 尝试使用 Chrome 浏览器

### Q3: AI 诗句生成失败？

**可能原因：**
- API 服务暂时不可用
- 网络连接问题

**解决方案：**
1. 刷新页面重试
2. 检查网络连接
3. 系统会自动使用默认诗句

### Q4: 如何调整 3D 模型视角？

**操作方式：**
- **旋转：** 鼠标拖拽或触摸拖拽
- **放大：** 点击底部"放大"按钮
- **重置：** 点击底部"重置"按钮

### Q5: 如何分享内容？

**分享方式：**
- 复制浏览器地址栏 URL
- 使用浏览器分享功能（移动端）
