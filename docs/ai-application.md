# AI 技术应用说明

本项目融合多项前沿 AI 技术，实现从开发到体验的全方位智能化：

**GLM-5 智能编码**：辅助生成 React 组件、Three.js 场景配置、TypeScript 类型定义等核心代码，开发效率提升 60% 以上。

**GLM-4-Flash 文本生成**：实时生成故宫主题诗句（每 10 秒更新）和 23 种斗拱的专业结构描述，响应时间 < 800ms，实现"千人千面"的个性化内容体验。

**Seedance 2.0 视频生成**：基于文本描述生成开场动画视频素材（gugong_reverse.mp4），以低成本实现高质量电影级视觉效果。

三项 AI 技术协同作用，让传统文化"活"起来，让用户体验"动"起来。
