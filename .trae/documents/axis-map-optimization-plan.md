# 中轴线地图界面优化计划

## 修改目标
1. 增加地图网格密度，使网格线分布更加细密
2. 优化配色方案，确保整体视觉效果协调统一
3. 调整图层层级关系，确保文字内容清晰可见

---

## 详细修改步骤

### 1. 增加网格密度

**当前状态**：
- `gridSize = 40` (网格间距40px)
- 垂直线和水平线间隔较大

**修改方案**：
- 将 `gridSize` 从 `40` 改为 `20`
- 增加次要网格线（更细的线条）形成双层网格效果
- 主网格线保持较粗，次网格线更细

**代码位置**：`generateGrid()` 函数（第586-610行）

### 2. 优化配色方案

**当前配色**：
```css
--bg-color: #faf9f7;        /* 米白色背景 */
--primary-color: #b91c1c;   /* 红色主色 */
--secondary-color: #78716c; /* 灰色辅助色 */
--grid-color: rgba(26, 26, 26, 0.06);
```

**优化方案**：
- 保持中国风基调，增强视觉层次
- 调整网格线颜色，使其更加细腻
- 增加金色点缀色，呼应故宫皇家气质
- 优化文字面板背景，增加与地图的对比度

**具体调整**：
```css
/* 优化后的配色 */
--bg-color: #faf8f5;           /* 更暖的米色 */
--primary-color: #9f1239;      /* 更深的故宫红 */
--secondary-color: #57534e;    /* 更深的辅助灰 */
--gold-accent: #a16207;        /* 金色点缀 */
--grid-color: rgba(87, 83, 78, 0.08);  /* 更明显的网格 */
```

### 3. 调整图层层级

**当前层级问题**：
- `.text-panel` z-index: 10
- `.location-marker` z-index: 10
- `.axis-line` z-index: 5
- `.grain` z-index: 1000
- 层级关系不够明确

**优化方案**：
```css
/* 明确的层级结构 */
.grain          z-index: 1000  /* 最顶层：噪点纹理 */
.progress-bar   z-index: 100   /* 进度条 */
.text-panel     z-index: 50    /* 文字面板（提高） */
.nav-hint       z-index: 40    /* 导航提示 */
.location-marker z-index: 30   /* 地点标记 */
.axis-line      z-index: 20    /* 中轴线 */
.grid-background z-index: 1    /* 网格背景（最底层） */
```

3. 文字面板始终清晰可见，不会被地图元素遮挡
