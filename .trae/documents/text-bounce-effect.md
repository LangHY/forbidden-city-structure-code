# 文字上浮回弹效果计划

## 问题分析

用户希望文字上浮动画有回弹效果，让动画更有活力和弹性感。

## 当前状态

```css
.text-label.animate, .text-title.animate, .text-subtitle.animate {
    animation: floatUp 0.7s cubic-bezier(0.65, 0, 0.35, 1) forwards;
}
```

**特点：** 有阻尼感，但没有回弹效果。

## 解决方案

### 方案：使用带回弹的贝塞尔曲线

**推荐贝塞尔曲线：**
```css
cubic-bezier(0.34, 1.56, 0.64, 1)
```

这个曲线特点：
- 结束时会超过目标位置
- 然后回弹到最终位置
- 有明显的弹性感
- 活泼有活力

### 备选方案

**方案A：back.out（GSAP）**
- GSAP内置的回弹曲线
- 效果类似

**方案B：自定义关键帧动画**
- 使用CSS关键帧实现多阶段回弹
- 更精细的控制

## 推荐配置

| 元素 | 时长 | 缓动曲线 |
|------|------|----------|
| text-label | 0.8s | cubic-bezier(0.34, 1.56, 0.64, 1) |
| text-title | 0.8s | cubic-bezier(0.34, 1.56, 0.64, 1) |
| text-subtitle | 0.8s | cubic-bezier(0.34, 1.56, 0.64, 1) |

## 实施步骤

### Step 1: 更新CSS动画缓动曲线
- 将 `cubic-bezier(0.65, 0, 0.35, 1)` 改为 `cubic-bezier(0.34, 1.56, 0.64, 1)`
- 将动画时长从 0.7s 改为 0.8s

### Step 2: 验证效果
- 确认文字上浮有回弹效果
- 确认回弹幅度适中，不过于夸张

## 预期效果

- 文字上浮到目标位置后会轻微超过
- 然后回弹到最终位置
- 有弹性感和活力感
- 动画更加生动有趣
