# 最优缓动曲线选择 Spec

## Why
当前使用的power2.inOut曲线效果不够优雅，需要找到最适合地图缩放和平移动画的缓动曲线。

## What Changes
- 测试多种专业级缓动曲线
- 选择最适合的曲线组合
- 调整动画时长配合曲线

## Impact
- Affected specs: elegant-smooth-animation
- Affected code: axis-map.html (JavaScript动画参数)

## 候选缓动曲线分析

### 1. power3.inOut（推荐）
- **特点**：三次方加速减速，比power2更平滑
- **曲线**：开始慢加速 → 中间匀速 → 结束慢减速
- **适用**：通用动画，优雅自然
- **时长建议**：缩放800ms，平移600ms

### 2. circ.inOut
- **特点**：圆形曲线，加速减速更明显
- **曲线**：类似圆弧，过渡非常平滑
- **适用**：需要更明显加速减速感的动画
- **时长建议**：缩放700ms，平移500ms

### 3. expo.inOut
- **特点**：指数级加速减速
- **曲线**：开始极慢 → 中间快 → 结束极慢
- **适用**：需要戏剧性效果的动画
- **时长建议**：缩放900ms，平移700ms

### 4. back.inOut（带回弹）
- **特点**：轻微回弹效果
- **曲线**：开始轻微后退 → 加速 → 结束轻微过冲
- **适用**：需要活力感的动画
- **时长建议**：缩放600ms，平移450ms

## 推荐方案

**power3.inOut + 适中时长**

| 动画 | 时长 | 缓动曲线 |
|------|------|----------|
| 缩放 | 800ms | power3.inOut |
| 平移 | 600ms | power3.inOut |
| 面板 | 500ms | CSS ease-in-out |

**优势：**
- 三次方曲线比二次方更平滑
- 加速减速过渡自然
- 不会感觉匆忙或拖沓
- 专业级动画效果

## ADDED Requirements

### Requirement: 使用power3.inOut缓动曲线
系统应使用三次方加速减速曲线。

#### Scenario: 缩放动画
- **WHEN** 执行缩放动画
- **THEN** 使用power3.inOut缓动曲线
- **AND** 动画时长800ms
- **AND** 加速减速过渡平滑

#### Scenario: 平移动画
- **WHEN** 执行平移动画
- **THEN** 使用power3.inOut缓动曲线
- **AND** 动画时长600ms
- **AND** 移动过程优雅自然
