# 文本逐行上浮动画 Spec

## Why
为故宫中轴线地图页面的文本内容添加逐行上浮动画效果，模拟文字从水中浮起的视觉效果，增强页面的艺术感和沉浸式体验，让用户在阅读历史信息时感受到"历史浮现"的意境。

## What Changes
- 为左侧文字面板的标题、副标题和正文内容添加逐行上浮动画
- 实现文本行按自然顺序依次触发动画
- 添加透明度变化和位置移动的组合动画效果
- 确保动画在主流浏览器中兼容

## Impact
- Affected specs: axis-map-page
- Affected code: `public/axis-map.html`

## ADDED Requirements

### Requirement: 逐行上浮动画效果
系统应为文本内容提供从下方向上浮起的动画效果：
- 动画起始状态：文本位于正常位置下方30px，透明度为0
- 动画结束状态：文本位于正常位置，透明度为1
- 动画持续时间：每行约0.6-0.8秒
- 缓动函数：使用ease-out或cubic-bezier实现自然的浮起效果

#### Scenario: 文本初始显示
- **WHEN** 用户滚动到新的地点位置，文本内容更新时
- **THEN** 文本行按顺序依次从下方浮起，模拟从水中浮现的效果

### Requirement: 顺序执行动画
动画应严格按照文本的自然行数顺序依次触发：
- 第一行先开始动画
- 后续每行在前一行开始后固定时间间隔（约100-150ms）开始动画
- 所有行动画完成后，文本完全清晰显示

#### Scenario: 多行文本动画序列
- **WHEN** 文本内容包含多行（如标题、副标题、多个段落）
- **THEN** 各行按从上到下的顺序依次浮起，形成流畅的视觉序列

### Requirement: 浏览器兼容性
动画效果应在以下浏览器中正常工作：
- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

#### Scenario: 跨浏览器显示
- **WHEN** 用户使用任何主流浏览器访问页面
- **THEN** 文本上浮动画效果一致，无明显性能问题或显示异常

### Requirement: 动画触发时机
动画应在以下时机触发：
- 页面首次加载时，初始文本执行上浮动画
- 用户滚动切换地点时，新文本内容执行上浮动画
- 动画执行期间，文本内容不可交互，直到动画完成

#### Scenario: 地点切换动画
- **WHEN** 用户滚动页面切换到新的地点
- **THEN** 旧文本淡出，新文本逐行上浮出现
